# PHP历史与发展

PHP，全称`PHP: Hypertext Preprocessor`，动态类型和弱类型的脚本语言，最大的特色是可以和HTML混合编写，因此极其适合编写动态网张，被广泛应用于Web后台开发领域，随着互联网的兴起，PHP迅速流行开来。

不像Go，Java等语言背后都有资金雄厚的公司在支持，PHP可能是目前流行的编程语言中少有的由社区驱动开发的编程语言。
