## 介绍

PHP其实是一门非常优秀的语言，语法易上手，简单实用，无需太多背景知识，就可以上手开发，在PHP7之后，性能更是得到极大的提升，当然PHP也有一些历史包袱和实用主义至上的问题，总之，在熟练使用PHP进行开发后，是非常有必要和值得深入地去学习和研究的。本项目将从基本语法，内核实现和应用实践等方面重新探索和梳理。
